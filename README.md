A console-based Wordle clone made in C++ for a class project.

## Run Locally

Clone the project

```bash
git clone https://github.com/aelew/wordle.git
```

Run the game

```bash
cd wordle/wordle
make && ./app
```

Run the keyboard viewer

```bash
cd wordle/keyboard
make && ./app
```
